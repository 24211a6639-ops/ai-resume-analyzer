# 🚀 AI Resume Analyzer

An AI-powered web app that analyzes resumes and matches them with job descriptions.

## 🔥 Features
- Resume parsing (PDF/DOCX)
- Skill extraction using NLP
- Match score calculation
- Missing skills detection
- Smart suggestions
- ATS score breakdown

## 🛠 Tech Stack
- Python
- Streamlit
- spaCy
- Scikit-learn
- Plotly
